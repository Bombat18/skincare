## 🧪 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/Bombat18/skincare.git
   cd skincare

   
````

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Build for production**

   ```bash
   npm run build
   ```
````
